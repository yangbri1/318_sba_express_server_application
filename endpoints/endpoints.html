API Documentation

Quick Start

Navigate to the homepage using path URL: http://localhost:3000/

Accessing Comments

  • To view all of the comments: http://localhost:3000/api/comments

  • To view a comment via their comment index id: 'http://localhost:3000/api/comments/{:id}'

  • Accessing posts made by an user's userId: 'http://localhost:3000/api/comments/users/{:userId}'

    • Getting Posts

  • To view all of the posts: http://localhost:3000/api/posts

  • Accessing post via their post index id: 'http://localhost:3000/api/posts/{:id}'

  • Accessing posts made by an user's userId: 'http://localhost:3000/api/posts/users/{:userId}'

    • Retrieving Users

  • To view all of the users: http://localhost:3000/api/users

  • To view the which user is linked to id: 'http://localhost:3000/api/users/{:id}'

  • To view the user's personal info, their comment and post history: 'http://localhost:3000/api/users/user/{:userId}'

    • Query Parameters

  • To view a selected number of comments: http://localhost:3000/api/comments/search?limit=2

  • To view a selected number of posts: http://localhost:3000/api/posts/search?limit=3

  • To view a selected number of users: http://localhost:3000/api/users/search?limit=1
