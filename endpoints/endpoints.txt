<h1>API Documentation</h1>
    <h2>Quick Start</h2>
    <p>Navigate to the homepage using path URL: 'http://localhost:3000/'</p>

    <h2>Accessing Comments</h2>
    <p><li>To view all of the comments: 'http://localhost:3000/api/comments'</li></p>
    <p><li>To view a comment via their comment index id: 'http://localhost:3000/api/comments/{:id}'</li></p>
    <p><li>Accessing posts made by an user's userId: 'http://localhost:3000/api/comments/users/{:userId}'</li></p>

    <h2>Getting Posts</h2>
    <p><li>To view all of the posts: 'http://localhost:3000/api/posts'</li></p>
    <p><li>Accessing post via their post index id: 'http://localhost:3000/api/posts/{:id}'</li></p>
    <p><li>Accessing posts made by an user's userId: 'http://localhost:3000/api/posts/users/{:userId}'</li></p>
    
    <h2>Retrieving Users</h2>
    <p><li>To view all of the users: 'http://localhost:3000/api/users'</li></p>
    <p><li>To view the which user is linked to id: 'http://localhost:3000/api/users/{:id}'</li></p>
    <p><li>To view the user's personal info, their comment and post history: 'http://localhost:3000/api/users/user/{:userId}'</li></p>
